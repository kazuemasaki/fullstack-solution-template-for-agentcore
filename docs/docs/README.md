# FAST Documentation

The Fullstack AgentCore Solution Template (FAST) provides a production-ready foundation for building full-stack applications with Amazon Bedrock AgentCore. This documentation covers deployment, configuration, and integration patterns.

![FAST Architecture](architecture-diagram/FAST-architecture-20251201.png)

## What You'll Find Here

**Deployment** - Instructions for deploying FAST to AWS, including prerequisites, configuration, and deployment commands.

**Agent Configuration** - Guidance on using existing agent patterns (Strands, LangGraph) or creating custom implementations.

**Gateway Integration** - Implementation details for AgentCore Gateway with Lambda targets, enabling scalable tool execution.

**Memory Integration** - Practical guide for integrating short-term and long-term memory capabilities into agents.

**Streaming** - How to implement real-time streaming responses from agents to the frontend.

**Code Interpreter** - Integration patterns for Amazon Bedrock AgentCore Code Interpreter.

**Version Management** - Checklist for version bumps and release procedures.

## Getting Started

New to FAST? Start with the Deployment guide to get your application running, then explore Agent Configuration to customize the behavior for your use case.
